# Welcome Manado - Client Frontend

Ini adalah repositori untuk frontend customer platform **Welcome Manado**.

## 🚀 Teknologi
- **Vue 3** (Composition API)
- **Vite**
- **Tailwind CSS 4**
- **Vue Router**
- **Axios**

## 🛠️ Cara Menjalankan
```bash
npm install
npm run dev
```
*Aplikasi akan berjalan di `http://127.0.0.1:5173`*

## 📡 Integrasi API
Konfigurasi API dapat ditemukan di `src/services/api.js`. Pastikan backend (`api_wm`) sudah berjalan.

*Untuk dokumentasi lengkap workspace, silakan lihat di folder `api_wm/README.md`.*
